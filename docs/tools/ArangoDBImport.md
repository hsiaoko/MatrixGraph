# ArangoDB Import for edgelistcsv2arangodbjson

## Overview

When users run `edgelistcsv2arangodbjson`, they still need to import the generated JSON/JSONL files into ArangoDB.

This page provides a robust line-by-line import script for large JSONL files.

## Input Files

Generated by `graph_converter`:
- `graph_structure.json`
- `pivot_graph_ids.jsonl`
- `pivot_graphs.jsonl`

## Robust Import Script

```bash
cat > import_robust.sh << 'EOF'
#!/bin/sh

# ================= 配置区域 =================
DB_NAME="stanford"
USER="root"
PASS="123456"
ARANGO_HOST="127.0.0.1"
ARANGO_PORT="8529"
BASE_URL="http://${USER}:${PASS}@${ARANGO_HOST}:${ARANGO_PORT}/_db/${DB_NAME}/_api/document"
# ===========================================

echo "🔍 检查 ArangoDB 连接..."
if ! curl -s --connect-timeout 3 "${BASE_URL}/_collection" -o /dev/null; then
    echo "❌ 无法连接到 ${BASE_URL}。"
    exit 1
fi
echo "✅ 连接成功！"

import_jsonl_robust() {
    FILE_PATH="$1"
    COLLECTION_NAME="$2"
    
    if [ ! -f "$FILE_PATH" ]; then
        echo "❌ 文件不存在: $FILE_PATH"
        return 1
    fi

    echo ""
    echo "🚀 开始稳健导入 (逐行): $FILE_PATH -> $COLLECTION_NAME"
    
    # 1. 确保集合存在
    curl -s -X POST "${BASE_URL}/_collection" \
         -H "Content-Type: application/json" \
         -d "{\"name\": \"${COLLECTION_NAME}\"}" -o /dev/null
    
    count=0
    error_count=0
    
    # 2. 逐行读取并立即发送 (绝不拼接)
    while IFS= read -r line || [ -n "$line" ]; do
        # 跳过空行
        case "$line" in
            "") continue ;;
        esac
        
        count=$((count + 1))
        
        # 每 5000 行打印一个点，避免刷屏
        if [ $((count % 5000)) -eq 0 ]; then
            echo -n "."
        fi

        # 🛠️ 核心逻辑：通过管道将单行数据传给 curl
        # 这样无论 line 多长、包含什么字符，都不会触发 "Argument list too long"
        # 也不会触发 "Unterminated quoted string"
        if ! printf '%s\n' "$line" | curl -s -X POST "${BASE_URL}/${COLLECTION_NAME}?waitForSync=false" \
             -H "Content-Type: application/json" \
             --data-binary @- -o /dev/null; then
            error_count=$((error_count + 1))
        fi
        
    done < "$FILE_PATH"
    
    echo ""
    echo "✅ 完成: $FILE_PATH"
    echo "   总行数: $count"
    if [ "$error_count" -gt 0 ]; then
        echo "   ⚠️  失败行数: $error_count (请检查日志)"
    else
        echo "   🎉 全部成功!"
    fi
}

# 执行导入
import_jsonl_robust "/ssd_data/zhuxk/arangodb_graphs/stanford/pivot_graph_ids.jsonl" "pivot_graph_ids"
import_jsonl_robust "/ssd_data/zhuxk/arangodb_graphs/stanford/pivot_graphs.jsonl" "pivot_graphs"

echo ""
echo "🎉 所有导入任务结束！"
EOF

chmod +x import_robust.sh
./import_robust.sh
```

## Notes

- Update DB connection and file paths before running the script.
- `graph_structure.json` is schema/meta data, while instance vertices/edges are in `pivot_graphs.jsonl`.
